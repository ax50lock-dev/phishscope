◉ PhishScope
Settings

These settings sync with the extension popup. Database lookups catch phishing on clean-looking domains that heuristics can't — each lookup sends the URL (or domain) to that service.

Display mode

Controls how results are explained in the popup.

Theme

Pick a color theme.

VirusTotal

Consensus from ~90 security vendors. Free key: virustotal.com → profile → API key.

PhishTank

Community database of verified phishing URLs. Works without a key at lower limits.

Domain age (RDAP)

Checks how recently the domain was registered — most phishing domains are days old. No key needed.

AI explanations

Optional. Uses your own Anthropic API key to narrate findings. Only findings are sent, never page content, and only when you click "Explain with AI".
